Connecting everyone to the internet

  • Essential VPN Connection Information
    • These are the critical details users need to connect to your VPN:

  • 1.1. Server Address
    • The public IP address or domain name of your VPN server. Example: 123.45.67.89 or vpn.example.com.

  • 1.2. Ports
    • Specify the open ports on your server. Common OpenVPN ports: 1194 (UDP), 443 (TCP). Ensure you list both TCP and UDP if you support both.

  • 1.3. Protocol
    • Mention the protocol supported: TCP: Reliable and secure but slower. UDP: Faster, recommended for most use cases.

  • 1.4. Authentication Details
    • Provide instructions for how users will authenticate: Username/Password (e.g., testuser / testpass). Certificates (link to .crt or .ovpn files, if required).

  • 1.5. VPN Configuration File
    • Offer ready-to-use .ovpn files for OpenVPN users. Include server address, protocols, ports, and certificate information.
